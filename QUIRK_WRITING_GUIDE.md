**[Behavior] + [Context] + [Personality trait]**

## Template Structure

### For Dogs:

```
"[Unique behavior] + [what makes them special]"

Examples:
- "Greets everyone with gentle 'hugs' using his front paws"
- "Loves to 'sing' along when the doorbell rings"
- "Has a collection of favorite sticks that he guards like treasures"
```

### For Cats:

```
"[Personality trait] + [adorable habit]"

Examples:
- "Curls up in a perfect circle and chirps softly when content"
- "Believes all boxes are thrones designed specifically for royalty"
- "Has mastered the art of the slow blink to show affection"
```

## Quick Test

Ask yourself:

1. **Is it specific?** (Not "friendly" but "greets everyone at the door")
2. **Is it unique?** (Not already covered by other fields)
3. **Is it endearing?** (Would it make someone smile?)
4. **Is it true?** (Based on actual observed behavior)
